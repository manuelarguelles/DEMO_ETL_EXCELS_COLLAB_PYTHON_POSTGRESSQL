📌 ETL Diario con Python y PostgreSQL
Este notebook realiza un proceso ETL diario, automatizado con Python (Pandas, SQLAlchemy, dotenv) en Google Colab, que incluye:

🔹 Extracción: Lectura y unión de múltiples archivos Excel diarios.

🔹 Transformación: Creación de columnas calculadas, integración con archivo maestro SKU y validación de datos.

🔹 Carga: Inserción de datos procesados en una base de datos PostgreSQL existente, utilizando variables de entorno para credenciales seguras.

Incluye pasos de validación intermedia y manejo básico de errores comunes.
